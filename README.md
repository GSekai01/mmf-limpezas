# Site da MMF Limpezas

Site institucional da MMF Limpezas, construído com Next.js, React, TypeScript e
Tailwind CSS.

## Desenvolvimento

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.

## Verificações

```bash
npm run lint
npm run build
```

## Estrutura

- `src/app/page.tsx`: página institucional
- `src/app/globals.css`: identidade visual e comportamento responsivo
- `src/app/layout.tsx`: fontes, metadados e configurações globais
- `public/`: logos e futuros arquivos públicos

## Próximas integrações

O canal de captura de leads será conectado em uma etapa separada. A página foi
estruturada para receber formulário, WhatsApp, email ou integração com CRM sem
alterar a arquitetura visual.
